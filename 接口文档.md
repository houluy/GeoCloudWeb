## 接口文档

## 1.1 用户反馈上传接口

-请求路径：/api/feedback

-请求方式：POST

-请求参数：

| 参数名 | 类型 | 是否必填 | 说明 |
| ----- | ---- | -------| --- |
| orderDataID | string | 是 | 用户订单ID |
| score       | string | 是 | 对本次服务的评分 |
| content     | string | 是 | 意见及建议 |

-返回参数：

| 参数名 | 类型 | 说明 |
| --- |
| code | int | 状态码 |
| message | string | 状态信息 |

-请求示例：

```
POST /api/feedback/upload HTTP/1.1
Host: 127.0.0.1:8080
Content-Type: application/json

{
    "OrderID": "1234567890",
    "score": "非常满意",
    "content": "服务很好，数据更新及时"
}
```

-返回示例：

```
HTTP/1.1 200 OK
Content-Type: application/json
示例成功响应：
{
    "success": true,
    "message": "反馈提交成功"
}
示例失败响应：
{
    "success": false,
    "message": "评分和内容不能为空"
}
```
